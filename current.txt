  Description  :
    ALL SAFE 2.0 is an intelligent, AI-powered cybersecurity platform built
    to detect and mitigate digital threats in real time. It unifies multiple
    threat intelligence engines, AI models (Gemini + LLaMA), and live

    both technical users and everyday internet users in India.

  3. TECH STACK
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  FRONTEND
  ─────────────────────────────────────────────
  • Framework     : React 18 + Vite
  • Animations    : Framer Motion
  • Icons         : Lucide React
  • HTTP Client   : Axios
  • 3D Globe Map  : react-globe.gl
  • PDF Reports   : jsPDF + jspdf-autotable
  • Fonts         : Syne, Space Grotesk, JetBrains Mono (Google Fonts)
  • Styling       : Pure CSS (CSS Variables, Glassmorphism, Dark Theme)

  BACKEND
  ─────────────────────────────────────────────
  • Framework     : FastAPI (Python 3.11)
  • Server        : Uvicorn (ASGI) with --reload
  • Database      : SQLite3 (allsafe.db — scan logs)
  • System Mon.   : psutil (CPU, RAM, local network connections)
  • PDF Parsing   : PyPDF2
  • DOCX Parsing  : python-docx
  • Phone OSINT   : phonenumbers library
  • WHOIS Lookup  : python-whois
  • Networking    : httpx (async HTTP client)
  • Env Config    : python-dotenv

  AI ENGINES
  ─────────────────────────────────────────────
  • Primary AI    : Google Gemini 2.5 Flash (via google-genai SDK)
  • Secondary AI  : LLaMA 3.3 70B (via Groq Cloud API)
  • Fallback      : If both fail, graceful error message shown to user

  4. ALL FEATURES (Complete List)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  [HOME / HERO]
  • Animated cybersecurity themed landing page
  • Live threat counter in the navbar (WebSocket-powered)
  • Global keyboard shortcuts (H=Home S=Scan M=Map etc.)
  • Command Palette (Ctrl+K) for quick navigation & scanning

  [OMNI SCANNER] ← NEW FEATURE
  • Single unified input box (URL / IP / Domain / Password / Text)
  • Auto-detects input type using regex + AI classification
  • Routes to the correct backend scan automatically
  • Displays: Threat Score (0–100), Risk Level, Reason, Recommendation
  • Animated threat meter / progress bar
  • Powered by Gemini AI for unclassified inputs

  [ENGINES SCANNER]
  • Scan Modes: URL, File Upload, Hash, Domain, IP, Phone, Email
  • VirusTotal: 70+ AV engine results with stats breakdown
  • IP Scan: VirusTotal + OTX + AbuseIPDB + ThreatFox + HoneyDB + WHOIS
  • Domain Scan: VirusTotal + OTX + ThreatFox cross-check
  • "Deep Web Threat Context" card showing all OSINT sources
  • AI Intelligence Briefing (Gemini-generated for every scan)
  • PDF Report Download (professional multi-section report)
  • Scan History logging to SQLite database
  • Idle state with animated visual when no scan is running

  [LIVE THREAT MAP]
  • Real-time 3D globe (react-globe.gl) with arc animations
  • Attack data from 6 simultaneous sources:
    1. ThreatFox API (active malware C2 IPs)
    2. AbuseIPDB API (crowd-reported abusive IPs)
    3. HoneyDB API (honeypot trap hits)
    4. AlienVault OTX (threat pulses)
    5. Local System Sensor (YOUR machine's live network connections)
    6. MaxMind Risk Engine (compromised IP feed from EmergingThreats)
  • WebSocket live feed (updates every 30 seconds)
  • 30+ categorized attack types (RECON, EXPLOIT, MALWARE, etc.)
  • DEFCON level indicator, attack stats sidebar, category filter
  • Country target rings + arc animations per attack

  [JOB SCAM DETECTOR]
  • Rule-based detection (12 scam pattern rules with weighted scoring)
  • AI-powered analysis (Gemini generates detailed risk assessment)
  • Scam Score 0–100 with Risk Level (CLEAN / LOW / MEDIUM / HIGH)
  • Document Upload: supports PDF, DOCX, TXT analysis
  • Domain extraction from uploaded documents + VirusTotal scan
  • Red Flag Checklist tab (manual interactive checklist)
  • Common scam examples tab for reference
  • PDF Report Download with full analysis

  [EMAIL HEADER ANALYZER]
  • Paste raw email headers for advanced analysis
  • Checks: SPF, DKIM, DMARC authentication
  • Detects: Header spoofing, relay chains, suspicious mail servers
  • Gemini AI generates a professional summary
  • PDF Report download

  [AI CHATBOT] ← MAJOR UPGRADE
  • Collapsible sidebar with conversation history
  • Multi-model support: Gemini 2.5 Flash + LLaMA 3.3 70B (switch live)
  • Streaming typewriter animation for AI responses
  • Natural Language Scan Triggering:
    - "Scan this URL: example.com" → auto runs URL scan
    - "Is 8.8.8.8 safe?" → auto runs IP scan  
    - "Check this job offer..." → auto runs Job Scam scan
    - "Get my security stats" → auto pulls system stats
    - "Show my recent scans" → auto pulls DB history
    - "Omni scan: <anything>" → auto routes through unified scanner
  • Inline Scan Result Cards (collapsible, color-coded by risk)
  • Full Markdown rendering (headers, bullets, code blocks, bold)
  • Capability pills overview on welcome screen
  • Copy button on every AI message
  • Animated glow avatar per model

  [INTELLIGENCE DASHBOARD]
  • Total scans, High/Medium/Clean breakdown
  • Live CPU, RAM, Disk usage (real system data via psutil)
  • Scan type distribution charts
  • Recent scan history preview

  [SCAN HISTORY / LOGS]
  • Full searchable scan log (SQLite backend)
  • Shows: type, target, risk level, timestamp
  • Clear history function

  [CYBER NEWS]
  • Live feed from The Hacker News (via RSS2JSON API)
  • Category tags + publication dates + article summaries
  • Click-through to full articles

  [CRYPTO VAULT]
  • Crypto price tracker (live data)
  • Wallet security tips

  [PASSWORD STRENGTH ANALYZER]
  • Real-time entropy calculation
  • Strength meter (Weak / Fair / Strong / Very Strong / Fortress)
  • Detects common passwords, keyboard patterns
  • Suggestions for improvement
